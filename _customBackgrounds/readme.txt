Custom videos must are activated in theme menu, with or without audio.


Here you can place a video named "default.mp4".

this will display on all systems not customized.

You can also place videos files named by theme name, examples: snes.mp4, megadrive.mp4 etc... to display different files in each system.

BECAREFULL, YOU MUST ACTIVATE BICHROMATIC MODE IF YOU DON'T USE BLACK AND WHITE VIDEOS..
