
CUSTOM VIDEOS or CUSTOM STATIC VIDEO must are activated in theme menu, with or without audio.


You can place a video named "default.mp4", this will display on all systems not customized 

when you select CUSTOM BACKGROUNDS AND CUSTOM BACKGROUNDS WITH AUDIO in menu.

For each systems you can use videos named by theme name, examples: snes.mp4, megadrive.mp4 etc... to display different files in each system.



You can use a video named "custom.mp4", this will display an unique background video in all theme and all systems 

when you SELECT CUSTOM STATIC AND CUSTOM STATIC WITH AUDIO in menu.



BECAREFULL, YOU MUST ACTIVATE BICHROMATIC MODE IF YOU DON'T USE BLACK AND WHITE VIDEOS..
